My Role

I am going to help Jorge with the backend side of the project, especially with making sure messages can be submitted, stored, displayed properly, and managed once the tree starts filling up. Since the project depends on user interaction, making sure the system works smoothly is important.

I also plan to give suggestions on the front end UI whenever needed, especially around layout, visual flow, and how users interact with the page. Even if my main focus is backend support, I still want to contribute to the overall design direction when helpful!

It's definitely a little difficult splitting work between four people because there are overlapping areas and multiple moving parts, but I think we will make it work by helping each other where needed and staying flexible throughout the process.
