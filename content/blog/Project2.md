## Kris' Contributions

For this Project, I focused on the front-end development. I designed and styled the homepage, image choice, fonts, and navigation of the website. I implemented responsive css to ensure that the project works across different screen sizes, styled key elements like about page, foot, navigation, and data labels. I also built a facts dropdown features. In addition, I refined spacing, typography, hover effects, and color choices such as matching the ocean and seismic labels to the corresponding lines to improve readability. 

## Technical Progress

Throughout this project, I spent a lot of time researching and experimenting with layout, layering, and translating data into a visual system. One of the proudest moments for me was figuring out the correct css for creating a unification for the title and the gif.

Another area I experimented with was how to include additional information without overcrowding the page. Initially, I placed the facts directly on the page, but it felt too heavy and distracted from the visual. This led me to experiment with a dropdown interaction. I tested a few approaches, including hover based CSS, but it was unreliable and did not work well on different devices. I ended up using a simple JavaScript toggle, which allowed the facts to stay hidden until the user chose to open them.

Overall, this project helped me better understand how design and code work together to create a experience rather than just a fuctional website. I also became more comfortable with troubleshooting issues and refining ideas through iteration. This project showed me how code can be used as a tool to translate complex systems into something more intuitive and engaging.
